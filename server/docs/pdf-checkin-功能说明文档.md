# PDF阅读打卡系统功能说明文档

## 一、系统概述

PDF阅读打卡系统是一套面向教师端和学生端的完整打卡管理解决方案，支持阅读、听力和视频三种任务类型，实现从资料管理、任务布置到学生打卡的全流程管理。

---

## 二、核心功能模块

### 2.1 资料库管理

**页面位置**：`/admin/material-library`

**功能描述**：
- 管理PDF课件和音频资料的文件夹与文件
- 支持PDF和音频两种类型的资料分类管理

**核心功能**：
1. **文件夹管理**
   - 新建文件夹
   - 删除文件夹
   - 文件夹优先展示（位于文件列表上方）

2. **文件管理**
   - 本地上传文件
   - 从云端选择文件
   - 编辑文件信息（名称、描述）
   - 删除文件
   - 纯文字列表展示（无封面图）

3. **数据持久化**
   - 使用 localStorage 保存文件夹和文件数据
   - 支持跨页面数据共享

---

### 2.2 课件生成

**页面位置**：`/admin/courseware-generate`

**功能描述**：
- 将PDF课件转换为可供学生阅读的学习材料

**核心功能**：
1. **一图多音**
   - 为每一页PDF生成对应的音频
   - 支持音频顺序播放
   - 音频与页面自动关联

2. **PDF分页处理**
   - 按页拆分PDF内容
   - 每页独立配置音频和说明

3. **预览功能**
   - 支持生成后的效果预览

---

### 2.3 课件编辑音频

**页面位置**：`/admin/audio-edit`

**功能描述**：
- 对已生成的课件进行音频编辑

**核心功能**：
1. **三栏布局**
   - 左侧：课件页面列表
   - 中间：音频配置
   - 右侧：音频播放控制

2. **音频交互**
   - 音频自动关联对应页面
   - 支持音频播放控制
   - 自动播放功能

3. **编辑功能**
   - 添加/删除音频
   - 调整音频顺序
   - 预览音频效果

---

### 2.4 任务设置

**页面位置**：`/admin/task-settings`

**功能描述**：
- 教师布置打卡活动的主页面
- 配置任务的基础信息

**页面布局**：
1. **顶部导航栏**
   - 返回箭头（返回上一页）
   - 页面标题："布置打卡活动"
   - "布置"按钮（右侧）

2. **表单内容区**
   - **任务名称**：文本输入
   - **打卡天数**：数字选择
   - **开始时间**：日期时间选择器
   - **结束时间**：日期时间选择器
   - **布置对象**：选择学生/班级
   - **任务内容**：点击添加任务

**隐藏内容**：
- 已删除"保存模板"按钮

**交互流程**：
```
点击"添加任务" 
  → 弹出任务类型选择对话框
  → 选择任务类型（阅读/听力/视频）
  → 保存基础信息到 localStorage
  → 根据类型跳转到不同页面
```

---

### 2.5 课件选择流程

**页面位置**：`/admin/textbook-select`

**功能描述**：
- 选择阅读或听力任务对应的课件资料

**页面布局**：
1. **顶部导航栏**
   - 返回箭头
   - 页面标题："选择课件"
   - 任务类型标签（红色=阅读，蓝色=听力）
   - "确定添加 (数量)"按钮

2. **内容区**
   - **上方：文件夹列表**
     - 只显示文件夹名称
     - 蓝色可点击链接样式
     - 鼠标悬停显示下划线
     - 无复选框、无文件数量、无操作列
     - 点击文件夹名称进入文件夹详情页

   - **下方：课件列表**
     - 支持多选（复选框）
     - 显示课件名称、类型、页数/时长
     - 文件夹和课件之间间距20px

**布局要求**：
- 无滚动条，内容自动撑开
- 优先展示文件夹，再展示课件
- 页面高度最小100vh

**样例数据**：
- 文件夹示例：语文课文、英语绘本、Reading AtoZ
- 课件示例：春-朱自清、The Cat in the Hat等

**交互逻辑**：
```
选择课件页面
  → 点击文件夹名称
  → 进入文件夹详情页
  → 勾选课件
  → 点击"确定添加"
  → 跳转到任务列表编辑页面
```

---

### 2.6 文件夹详情页

**页面位置**：`/admin/folder-detail`

**功能描述**：
- 查看文件夹内的所有课件
- 支持从不同入口进入，显示不同的交互模式

**智能识别来源**：
- 从"选择课件"进入：显示选择模式
- 从"资料库管理"进入：显示管理模式

**页面布局**：
1. **面包屑导航**
   - 动态显示来源（选择课件 / 资料库管理）
   - 当前文件夹名称

2. **顶部操作栏**
   - 文件夹图标
   - 文件夹名称
   - 文件数量标签
   
   **选择模式按钮**：
   - "确定添加 (数量)"按钮（蓝色）
   - "返回"按钮
   
   **管理模式按钮**：
   - "新建课件"按钮（蓝色）
   - "返回"按钮

3. **文件列表**
   - **选择模式**：带复选框，可多选
   - **管理模式**：无复选框，显示编辑、查看、移出操作

**交互功能**：
- 选择模式下勾选课件
- 点击"确定添加"跳转到任务列表编辑页面
- 点击"返回"根据来源智能返回

---

### 2.7 视频上传页面

**页面位置**：`/admin/video-upload`

**功能描述**：
- 上传视频任务的文件

**功能特性**：
1. **上传方式**
   - 本地上传：拖拽上传，限制500MB

2. **视频信息**

   - 视频描述

3. **上传流程**
   - 选择文件
   - 填写信息
   - 确认上传
   - 跳转到资料库管理页面

---

### 2.8 任务列表编辑页面

**功能描述**：

## 三、任务类型说明

系统支持三种任务类型：

### 3.1 阅读任务（Reading）
- **图标**：文档图标
- **颜色**：红色
- **来源**：PDF课件
- **展示**：显示页数

### 3.2 听力任务（Audio）
- **图标**：耳机图标
- **颜色**：蓝色
- **来源**：音频文件
- **展示**：显示时长

### 3.3 视频任务（Video）
- **图标**：视频图标
- **来源**：视频文件
- **上传方式**：本地上传或云端选择
- **限制**：500MB

## 四、数据流转设计

### 4.1 localStorage 数据表

| Key | 用途 | 数据结构 |
|-----|------|---------|
| `materialFolders` | 资料库文件夹和文件 | {pdf: [], audio: [], pdfFiles: [], audioFiles: []} |
| `tempTaskData` | 临时任务配置 | {任务名称, 打卡天数, 时间, 对象} |
| `selectedCourseware` | 选中的课件 | {taskType, files: []} |

### 4.2 页面跳转路由

| 来源页面 | 目标页面 | 触发条件 | 传递参数 |
|---------|---------|---------|---------|
| task-settings | textbook-select | 选择阅读/听力 | taskType |
| task-settings | video-upload | 选择视频 | taskType |
| textbook-select | folder-detail | 点击文件夹 | folderId, folderName, type, fromSelect, taskType |
| folder-detail | task-content-config | 确认添加 | taskType |
| folder-detail | textbook-select | 返回（选择模式） | taskType |
| video-upload | material-library | 上传完成 | - |

## 六、技术实现要点

### 6.1 前端技术栈
- **框架**：Vue 3 + Composition API
- **UI库**：Element Plus
- **路由**：Vue Router
- **构建工具**：Vite
- **语言**：TypeScript

### 6.2 关键组件
- `el-table`：表格展示（支持多选、斑马纹、边框）
- `el-card`：卡片容器
- `el-tag`：标签（任务类型标识）
- `el-button`：按钮（主按钮、链接按钮）
- `el-breadcrumb`：面包屑导航
- `el-empty`：空状态展示

### 6.3 响应式数据
- `ref()`：基本类型响应式
- `reactive()`：对象类型响应式
- `computed()`：计算属性
- `onMounted()`：生命周期钩子

### 6.4 路由传参
- **query参数**：用于页面间数据传递
- **路由监听**：watch route.query 变化
- **智能返回**：根据来源决定返回目标

## 七、开发注意事项

### 7.1 常见陷阱

1. **Vue组件结构完整性**
   - 编辑后验证闭合标签
   - 确保 template/script/style 完整

2. **变量重复声明**
   - 重构时清理旧代码
   - 避免同作用域重复声明

3. **Vue3 ref变量遮蔽**
   - 避免在生命周期内遮蔽顶层ref变量

4. **属性值引号**
   - Vue模板中属性值禁止直接使用双引号

### 7.2 数据持久化

- 使用 localStorage 而非后端API（当前阶段）
- 数据格式统一为JSON
- 添加错误处理（try-catch）

### 7.3 页面删除流程

删除页面需同步修改：
1. 路由配置（router/index.ts）
2. 侧边栏菜单（views/Sidebar.vue）
3. 未使用的图标导入
