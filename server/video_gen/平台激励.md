# 主流短视频平台 · 创作激励统计

> **用途**：为本项目（9:16 竖屏、约 4:20–5:20 科普成片）选型分发与算账参考。  
> **口径**：政策变动频繁，**千播单价均为社区/媒体估算或本项目中性假设**，非官方承诺；开通条件以各平台创作者中心为准。  
> **关联**：[需求.md](./需求.md) §9.4（B 站盈亏）、§13.4（多平台分发）。

| **E · 弱相关** | 微视、皮皮虾、好看视频等 | 无稳定全民激励或份额极低 | — | — | 不推荐作为主分发 | — |

**结论（本项目）**

- **算账与 §11 回流主站**：**B 站**（已写入需求 §9.4）。  
- **同片增量分发**：**视频号、支付宝、抖音、快手**优先；**小红书**待蒲公英分成正式化后上调。  
- **头条/西瓜/微博/百家号**：可作补充，但规格与画像与成片不完全一致，收益需实测。

## 2. 分平台说明

### 2.1 B 站（哔哩哔哩）

| 项 | 内容 |
| --- | --- |
| **激励名称** | [创作激励计划](https://www.bilibili.com/opus/77399369418311052)（含视频基础激励、广告分成、活动/爆款小目标等） |
| **怎么赚** | 非纯播放量 CPM；综合**喜爱度（赞投藏）、流行度、垂直度**；商业推广稿不计入 |
| **门槛** | 创作力/影响力 ≥55，信用分 ≥80 等 |
| **2024–2025 变化** | 扶持「近半年月均平台收入 &lt;5000 元」个人 UP；**视频基础激励月上限 2000 元** |
| **千播参考** | 社区波动大；本项目 `选题.md` 用 **2.3 元/千播** 做中性测算 |
| **成片适配** | 4～5 分钟竖屏科普与 B 站用户习惯匹配 |
| **API** | ✅ B 站开放平台投稿 + 数据统计 |

### 2.2 抖音（含西瓜同步）

| 项 | 内容 |
| --- | --- |
| **激励名称** | **抖音创作者伙伴计划**（2024 年 9 月起替代「中视频伙伴计划」等旧收益） |
| **怎么赚** | 优质原创视频收益，与**播放时长、质量、受众**相关；西瓜/头条需按各自入口同步规则 |
| **门槛** | 实名、近 90 天无违规、有效粉丝与作品质量等（常见说法 **有效粉丝 ≥5 万**，以 App 内为准） |
| **千播参考** | 无官方固定单价；不适合用单一 CPM 做盈亏表 |
| **成片适配** | 9:16 竖屏可发；科普需过审；**伙伴计划门槛高**，新号可能先无播放收益 |
| **API** | ✅ 上传 + `create_video`（`video.create` + OAuth） |

**西瓜 / 中视频**：旧「中视频伙伴计划」新入驻通道已关闭；横屏 ≥1min 收益并入**伙伴计划 / 头条视频创作收益**体系，发布入口必须正确同步，否则无收益。

## 3. 与本项目 MVP 的对应关系

| 优先级 | 平台 | 建议动作 |
| --- | --- | --- |
| P0 | B 站 | Worker 自动投稿 + 手动/定时回流（需求已定） |
| P1 | 视频号、支付宝 | 成片下载后手传（或二期 API）；开通分成计划需先攒粉 |
| P2 | 抖音、快手 | API 接入同 §13.4；新号先确认是否已开通伙伴计划/光合任务 |
| P3 | 小红书 | 观察蒲公英视频分成正式规则后再接 API |
| P4 | 头条、微博、百家号 | 有余力再测，不纳入首月 15 条盈亏表 |

**重要**：同一 MP4 多平台分发时，**仅 B 站数据**用于需求文档 §11 默认调参；其他平台收益单独记账，避免用未验证的千播假设指导选题。

## 4. 不含或弱「创作激励」的说明

以下**不算**本项目意义上的「主流短视频创作激励」主战场，故不展开：

- **纯直播打赏/带货**（各平台均有，与成片流水线无关）  
- **短剧厂牌分账**（优酷、红果、快手磁力新剧等）——合同制，非批量科普  
- **微视、皮皮虾、好看视频**等——用户与激励规模弱  
- **海外 TikTok Creator Rewards**——需独立账号与合规，不在国内方案内
